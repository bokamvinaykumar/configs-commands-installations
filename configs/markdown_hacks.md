 
Headings 
========================

```
H1
======================== 
H2     
------------------------  

```

>output

H1  
======================== 
H2     
------------------------

or 

```
# h1 Heading
## h2 Heading
### h3 Heading
#### h4 Heading
##### h5 Heading
###### h6 Heading
```

>output

# h1 Heading
## h2 Heading
### h3 Heading
#### h4 Heading
##### h5 Heading
###### h6 Heading

#Paragraphs

>Paragraphs are separated by empty lines. To create a new paragraph, press enter twice.
```
Paragraph 1
Paragraph 2
```

>output

Paragraph 1

Paragraph 2

#Character styles

```
*Italic characters* 
_Italic characters_
**bold characters**
__bold characters__

```

>output

*Italic characters*         
_Italic characters_     
**bold characters**     
__bold characters__     
     

#Ordered list

```$xslt
1.  item 1
2.  item 2
3.  item 3

```

>output 

1.  item 1
2.  item 2
3.  item 3

#Unordered list

```
* item 1
* item 2
* item 3


+ item 4
+ item 5
+ item 6

```


>output

* item 1
* item 2
* item 3

+   item 4
+   item 5
+   item 6
